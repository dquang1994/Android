Step 1: Drag web view into layout

Step 2: Get reference to web view: (inside OnCreate and must has final)


import android.webkit.WebView;

final WebView webView = (WebView) findViewById(R.id.WebView);

Step 3: Add permission into Manifest


<uses-permission android:name="android.permission.INTERNET" />

Step 4: Browse to url:


webView.loadUrl("http://www.cs.yale.edu/homes/tap/Files/hopper-story.html");
