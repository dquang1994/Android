Show the log on screen using:


import android.widget.Toast

Toast.makeText(activity, "text to show", duration).show();

Argument:

  • activity: the class that extern Activity
  • "text to show": the text you want to show on the screen
  • duration: Toast.LENGTH_SHORT or Toast.LENGTH_LONG

Show the log on logcat using:


import android.util.Log;

Log.i("info", "some info");

The output has format: info: some info
