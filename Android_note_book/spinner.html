Drag spinner into layout

Create string array for value of spinner in strings.xml file:


<string-array name="spinner_options">
	<item>0</item>
	<item>1</item>
	<item>3</item>
	<item>5</item>
	<item>10</item>
</string-array>

Select spinner mode, default (not select) is dropdown


android:spinnerMode="dropdown"

Fill entries property the name of array defined before in strings.xml file:


android:entries="@array/spinner_options"

Get reference to spinner:


import android.widget.Spinner;

Spinner spinner = (Spinner) findViewById(R.id.spinner);

setOnItemSelectedListener:


import android.view.View;
import android.widget.AdapterView;

spinner.setOnItemSelectedListener(new AdapterView.OnItemSelectedListener() {
	@Override
	public void onItemSelected(AdapterView<?> parentView, View selectedItemView, int position, long id) {
		//Do something when an item was selected
		String temp = String.valueOf(spinner.getSelectedItem());
		repeats = Integer.valueOf(temp);
	}

	@Override
	public void onNothingSelected(AdapterView<?> parentView) {
	}
});
